\# Hybrid Experimental

The goal of the Hybrid Experimental program is to develop an in-house model used to develop hybrid rocket engines at UTA Aeromavs.



This repository contains the design programs for the engine

The attached Wiki serves as the technical documentation for the project

The OnShape has 3D and parametric Models

The Google Drive has other miscillanious files
